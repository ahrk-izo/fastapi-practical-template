# FastAPI Practical Template

A practical FastAPI project template with Docker, testing, linting, and CI.

## Purpose

This repository is a practical FastAPI project template designed for small backend services.

It focuses on:

- Clear project structure
- Local development with Docker
- Testing with pytest
- Linting with Ruff
- Continuous integration with GitHub Actions
- Environment-based configuration

## Tech Stack

- Python
- FastAPI
- Docker
- pytest
- Ruff
- GitHub Actions
- uv

## Planned Features
